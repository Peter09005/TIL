getBytes 에 인코딩 설정 안넣어주면 default 인코딩 세팅인 utf-8로 바이트 반환해준다. 

```java
public static String toString(byte[] a) {  
    if (a == null)  
        return "null";  
    int iMax = a.length - 1;  
    if (iMax == -1)  
        return "[]";  
  
    StringBuilder b = new StringBuilder();  
    b.append('[');  
    for (int i = 0; ; i++) {  
        b.append(a[i]);  
        if (i == iMax)  
            return b.append(']').toString();  
        b.append(", ");  
    }  
}
```

자바에서의 byte는 MSB가 1이면 음수로 판단한다. 

참고로 자바에서는 음수를 2의 보수로 계산 한다 

#### 2의 보수 
```
int a = binary(010110)

int -a = 101001 + 1 -> 101010 

a - a -> 010110
.        101010
		 ------
	     000000
	     
2의 보수로 처리하면 나중에 연산을 할때 편함
0의 중복도 사라지게 되서 논리적임
```
